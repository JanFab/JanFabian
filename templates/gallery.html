Data science and Artificial intelligence

During my studies in Oulu I had opportunity to work on projects using latest approaches in Data science, such as AI.

Anime face generation - Generative Adversial Network

Part of the "Deep learning" course was development and tuning of GAN, capable to generate original images of Anime characters. Part of the GitHub is also a paper, documenting the work.

Link to GitHub.

Human exercise classification based on Multi-modal data

Multi Modal Data Fusion project was developed in cooperation with fellow student Aleksander Madajczak. Task was to classify human exercise based on data from depth camera and accelerometer. Multiple modalities were merged using multiple approaches from simple classifiers to Neural network.

Link to GitHub.

Affective computing - Emotion detection

University project was part of Affective Computing course. The background and challenges of emotion detection from visual and audio modalities were handled. Edge detection, feature extraction, principal component analysis were included.

Link to GitHub.
